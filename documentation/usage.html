To use cardlib, you first need to Install it.
Then you can import it into your python script.

Usage example

from cardlib.card import Card
from cardlib.deck import Deck
card1 = Card("Question", "Answer")
card2 = Card("Question2", "Answer2")
deck = Deck([card1, card2])
deck.shuffle()
randcard=deck.get_random_card()
print(randcard.question)
print(randcard.answer)
