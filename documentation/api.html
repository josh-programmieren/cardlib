cardlib.card.Card

Constructor: Card(front,back)
front: string, the question
back: string, the answer
card.to_dict(): returns a dictionary with the question and answer
Card.from_dict(dict:dict): returns a card from a dictionary
Param type: dict returns: Card

cardlib.deck.Deck

Constructor: Deck(cards:list)
cards: list of cards
deck.shuffle(): shuffles the deck
deck.serialize(filename:str): serializes the deck to a file
Param type: str returns: None
Deck.deserialize(filename:str): deserializes the deck from a file
Param type: str returns: Deck
deck.get_random_card(): returns a random card from the deck
Param type: None returns: Card
